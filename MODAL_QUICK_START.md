# ⚡ Project Modal - Quick Start Guide

## 🎯 **HOW IT WORKS**

### **Opening the Modal**
Click on **any project card** in the Projects section to open the details modal.

### **Viewing Details**
The modal shows:
- 📸 Project image/mockup
- 💻 Technology stack
- 🏗️ Architecture pattern
- 📝 Detailed description
- ✨ Key features
- 🔗 Action buttons (Demo & Source)

### **Closing the Modal**
Four ways to close:
1. **Click the X button** (top-right corner)
2. **Click outside** the modal (on the dark area)
3. **Press ESC** key on keyboard
4. **Click "View Details"** button again

## 📞 **NEED HELP?**

### **Common Questions**

**Q: How do I add real project images?**
A: Place images in `public/assets/images/projects/` and update the project data.

**Q: How do I change the demo/source links?**
A: Edit the `demoUrl` and `githubUrl` in the project data mapping.

**Q: Can I customize the modal colors?**
A: Yes! See [PROJECT_MODAL_GUIDE.md](PROJECT_MODAL_GUIDE.md) for customization options.

**Q: How do I add more features to a project?**
A: Edit the `getProjectFeatures()` function in `Projects.tsx`.

**Q: Can I add more projects?**
A: Yes! Add to the translations in `locales/en.json` and `locales/ar.json`.
