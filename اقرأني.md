## 🎨 التخصيص

### تحديث المعلومات الشخصية
1. افتح `src/locales/en.json` للإنجليزية
2. افتح `src/locales/ar.json` للعربية
3. عدّل المعلومات المطلوبة
4. احفظ الملف - التحديث تلقائي!

### تغيير الألوان
```tsx
// ابحث عن هذا في أي ملف Component:
from-blue-600 to-cyan-500

// غيّره إلى:
from-purple-600 to-pink-500  // بنفسجي-وردي
from-emerald-600 to-teal-500 // أخضر-تركوازي
```
