## 🌍 نظرة عامة | Overview

موقع شخصي احترافي عالمي المستوى (Portfolio) مصمم ومبني خصيصاً لعرض الأعمال والخبرات البرمجية والهندسية. تم بناء المشروع من الصفر باستخدام أحدث معايير الويب لتقديم تجربة مستخدم (UX/UI) سلسة، سريعة، ومتجاوبة بالكامل مع مختلف الشاشات.

> **رابط المعاينة المباشر:** [https://aladnaniazzam.github.io/portfolio/](https://aladnaniazzam.github.io/portfolio/)

---

## ✨ المميزات الرئيسية | Core Features

### 🌍 دعم كامل وعميق للغتين (i18n & RTL)
- **العربية والإنجليزية:** تحويل كامل وشامل لجميع النصوص بضغطة زر.
- **انعكاس التصميم التلقائي (RTL/LTR Layout):** إعادة هيكلة الواجهات بالكامل بما يتوافق مع اللغة المختارة.
- **تغيير الخطوط الديناميكي:** استخدام خطوط فاخرة ومتوافقة تلقائياً (`Tajawal`/`Cairo` للعربية ↔ `Inter`/`Poppins` للإنجليزية).
- **حفظ الحالة:** تذكر لغة المستخدم المفضلة تلقائياً عبر المتصفح.

### 🎨 نظام الثيمات الذكي (Dark & Light Mode)
- **الوضع الداكن (الافتراضي):** تصميم فاخر ومريح للعين مع ألوان متناسقة وعصرية.
- **الوضع الفاتح:** إضاءة ممتازة ووضوح تباين عالٍ للقراءة المريحة.
- **تأثيرات سلسة:** تبديل فوري وانسيابي بدون أي وميض (Flicker-free) عند إعادة التحميل.

### 📱 الاستجابة والأداء العالي (Performance & Responsiveness)
- **متجاوب 100%:** مخصص ومحسّن بدقة لكل من (الهواتف الذكية، الأجهزة اللوحية، والشاشات المكتبية الضخمة).
- **بناء قياسي خفيف:** حجم كود لا يتعدى الـ **300 كيلوبايت** بفضل البناء الذكي القياسي لـ Vite، مما يضمن سرعة تحميل فائقة.

## 🛠️ الترسانة التقنية | Tech Stack

- **الوجهة الأمامية:** React 19 (أحدث إصدار مستقر)
- **لغة البرمجة:** TypeScript (لضمان كود آمن وخالٍ من الأخطاء العشوائية)
- **المُنسق والمحرك:** Tailwind CSS v4 (أحدث تقنيات التصميم السريع)
- **أداة البناء (Bundler):** Vite 7 (لأعلى كفاءة في التطوير والبناء المستقر)
- **الأيقونات:** Lucide React
- **إدارة الحالة:** React Context API

## 🎯 أقسام الموقع | Portfolio Architecture

1. **الرئيسية (Hero Section):** واجهة ترحيبية ديناميكية تخطف الأنظار مع أزرار مباشرة لاتخاذ إجراء (Call to Action).
2. **نبذة عني (About Me):** ملخص احترافي يستعرض المسار الأكاديمي والشهادات بمرتبة الشرف.
3. **المهارات (Skills Matrix):** مصفوفة تفاعلية متقدمة تغطي أكثر من 50 مهارة برمجية مقسمة على 8 فئات هندسية.
4. **الخبرات المهنية (Experience Timeline):** خط زمني تفاعلي يستعرض التدرج الوظيفي والمشاريع المستقلة.
5. **المشاريع المميزة (Projects Showcase):** معرض استعراض بطاقات المشاريع مع تفاصيل روابط الأكواد والمعاينة.
6. **تواصل معي (Contact Form):** نموذج اتصال حديث ومقنن للتواصل السريع والمباشر.

## 🚀 التشغيل المحلي والبدء السريع

### المتطلبات الأساسية
- تثبيت بيئة **Node.js** (الإصدار 18 أو أحدث).

### خطوات التثبيت والتشغيل
```bash
# 1. استنساخ المستودع
git clone [https://github.com/AladnaniAzzam/portfolio.git](https://github.com/AladnaniAzzam/portfolio.git)

# 2. الدخول إلى مجلد المشروع
cd portfolio

# 3. تثبيت المكتبات الاعتمادية
npm install

# 4. تشغيل سيرفر التطوير المحلي
npm run dev
