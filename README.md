# Azzam Adnan - Personal Portfolio Website

A world-class, modern, and ultra-professional personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## 🌟 Features

### Design & User Experience
- **Premium Tech-Forward Design**: Clean, minimal, and highly professional enterprise feel
- **Dark/Light Mode**: Smooth theme transitions with system preference detection
- **Glassmorphic Elements**: Modern UI with frosted glass effects
- **Micro-interactions**: Subtle animations and hover effects
- **Smooth Scrolling**: Seamless navigation between sections
- **Entrance Animations**: Elegant fade-in and slide-in effects

### Internationalization (i18n)
- **Full Bilingual Support**: English (LTR) and Arabic (RTL)
- **Seamless Language Switching**: Toggle between languages with automatic:
  - Text direction change (LTR ↔ RTL)
  - Font family switching (Inter/Poppins ↔ Tajawal/Cairo)
  - Layout mirroring for perfect RTL experience
- **Persistent Preferences**: Language and theme choices saved to localStorage

### Responsive Design
- **Mobile-First Approach**: Optimized for all devices
- **Breakpoint Support**: 
  - Mobile (< 640px)
  - Tablet (640px - 1024px)
  - Desktop (> 1024px)
- **Adaptive Layouts**: Flexbox and Grid for perfect responsiveness
- **Touch-Optimized**: Enhanced mobile navigation

### Technical Architecture
- **Clean Code**: Semantic HTML5, component-based architecture
- **TypeScript**: Type-safe development
- **Context API**: Global state management for theme and language
- **Custom Hooks**: Reusable logic for theme and language
- **Performance**: Optimized builds with Vite

## 📋 Sections

1. **Hero Section**: Eye-catching introduction with CTA buttons
2. **About Me**: Education, certifications, and professional summary
3. **Skills Matrix**: Interactive categorized technical skills showcase
4. **Experience Timeline**: Professional journey with achievements
5. **Featured Projects**: Portfolio of production-ready applications
6. **Contact Form**: Interactive form with direct contact information
7. **Footer**: Professional closing with language support notice

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Customization

### Colors
The color palette uses:
- **Primary**: Deep Tech Blues/Slate
- **Accent**: Emerald Green and Cyber Cyan
- **Text**: Clean Grayscale

To customize, edit the Tailwind configuration or use the predefined gradient classes.

### Fonts
- **English**: Inter, Poppins (Google Fonts)
- **Arabic**: Tajawal, Cairo (Google Fonts)

### Translations
Edit the translation files to customize content:
- `src/locales/en.json` - English content
- `src/locales/ar.json` - Arabic content
