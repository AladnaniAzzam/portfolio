### **4. نموذج اتصال متطور** 📝

#### **التحسينات الجديدة:**

**A. تخطيط أفضل:**
```
Grid Layout (Desktop):
┌──────────────┬──────────────┐
│ Name Input   │ Email Input  │  ← جنباً إلى جنب
└──────────────┴──────────────┘
┌─────────────────────────────┐
│ Subject Input               │  ← عرض كامل
└─────────────────────────────┘
┌─────────────────────────────┐
│ Message Textarea (6 rows)   │  ← مساحة أكبر
└─────────────────────────────┘
```

**B. التحقق من الصحة (Validation):**
- ✅ **اسم**: يجب أن لا يكون فارغاً
- ✅ **بريد**: 
  - يجب أن لا يكون فارغاً
  - صيغة صحيحة (user@domain.com)
- ✅ **موضوع**: يجب أن لا يكون فارغاً
- ✅ **رسالة**: 
  - يجب أن لا تكون فارغة
  - على الأقل 10 أحرف

**C. رسائل الخطأ:**
```tsx
// بالإنجليزية:
"Name is required"
"Invalid email format"
"Message must be at least 10 characters"

// بالعربية:
"الاسم مطلوب"
"صيغة البريد الإلكتروني غير صحيحة"
"الرسالة يجب أن تكون 10 أحرف على الأقل"
```

**D. مؤشرات مرئية:**
- ✅ **علامة * حمراء** للحقول المطلوبة
- ✅ **حدود حمراء** عند وجود خطأ
- ✅ **رسالة تحذير** تحت الحقل مع أيقونة ⚠
- ✅ **Placeholder واضح** في كل حقل

**E. زر الإرسال:**
```tsx
// حالة عادية:
[📤 Send Message] ← تدرج ثلاثي الألوان

// حالة الإرسال:
[⏳ Sending...] ← مؤشر دوار + معطل

// بعد الإرسال:
✅ رسالة نجاح خضراء في الأعلى
```

**F. حالات الزر:**
- **عادي**: تدرج أزرق-سماوي-أخضر
- **Hover**: يرتفع + ظل أقوى
- **جاري الإرسال**: 
  - مؤشر دوران متحرك
  - نص "Sending..." / "جاري الإرسال..."
  - معطل (disabled)
- **بعد النجاح**: يعود للحالة العادية
